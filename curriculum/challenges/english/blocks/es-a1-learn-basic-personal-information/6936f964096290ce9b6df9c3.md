# --description--

To ask someone if they want something, Spanish uses the verb `querer`.

The verb form `quieres` is used with `tú` (you) to ask about another person's wishes or preferences. It is common in everyday conversation. For example:

- `¿Quieres ayuda?` - Do you want help?

- `¿Quieres más tiempo?` - Do you want more time?
